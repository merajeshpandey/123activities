## â“ Troubleshooting

**Q: My custom activities aren't showing up**
- Make sure you saved them
- Check the age group matches your game setting
- Custom activities appear randomly (50% chance when available)

**Q: AI isn't generating activities**
- Verify API key is correct
- Check you have API credits with your provider
- Look in browser console (F12) for errors
- Game will use fallback activities automatically

**Q: History isn't recording**
- Make sure you're clicking "Next Player" after each turn
- Check browser console for errors
- Try clearing cache and reloading

**Q: Layout not changing**
- Click the button multiple times to verify
- Refresh page to see if preference saved
- Try clearing localStorage and setting again

**Q: Buttons not visible on mobile**
- Try portrait mode
- Zoom out if needed
- Some buttons may scroll if screen is very small
